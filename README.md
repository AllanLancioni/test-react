## Requisitos

Devem ser criadas 3 telas:
- Login
- Home
- Ajuda

### Login

A tela de login deve apresentar dois campos:
- __Usuário:__ Campo `text` com no máximo `128 Bits` que valida se o valor preenhido é um `email`.
